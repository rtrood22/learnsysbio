Module 6: Y Protein Stability (Simple Gene Network)

Learn Systems Biology by UCONN CSMLab



  1. Modify the JavaScript code below
  2. Check/uncheck checkboxes
  3. Press [Run]
N = 250; // total number of data points x = new Array(N); // x protein concentration (uM) y = new Array(N); // y protein concentration (uM) d = new Array(N); // disturbance affecting y protein X = 10; // constant x protein concentration x[0] = X; // initial x protein concentration y[0] = 0; // initial y protein concentration pXY = 0.5; // production parameter pY = 0.9; // degradation/dilution parameter for (n = 1; n < N; n++) // N-1 loops (1 loop = 1 min) { x[n] = X; // constant x protein concentration if (n == 50){d[n] = 20;} // add disturbance 20 to y protein else if (n == 150) {d[n] = -20;} // at time 50 and disturbance -20 to else d[n] = 0; // y protein at time 150 y[n] = pXY*x[n-1] + pY*y[n-1] + d[n]; // update y protein concentration } // end of for loop
