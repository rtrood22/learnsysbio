Module 7: Extrinsic (Global) Noise Added to X Protein

Learn Systems Biology by UCONN CSMLab



  1. Modify the code below
  2. Check/uncheck checkboxes
  3. Press [Run]
N = 200; // total number of data points x = new Array(N); // x protein concentration (uM) y = new Array(N); // y protein concentration (uM) X = 30; // constant x protein concentration x[0] = X; // initial x protein concentration y[0] = 0; // initial y protein concentration pXY = 0.2; // production parameter pY = 0.9; // degradation/dilution parameter for (n = 1; n < N; n++) // N-1 loops (1 loop = 1 min) { //add extrinsic noise to x protein x[n] = X + 2*random.Gaussian(); // random.Gaussian() generates // a normally distributed random // number with mean 0 and standard // deviation 1 (extrinsic noise) y[n] = pXY*x[n-1] + pY*y[n-1]; // update y protein concentration } // end of for loop
